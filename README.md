# Shubham Singh Kalyanwat - Portfolio V2

Modern, tech-focused portfolio built with Next.js, Tailwind CSS, shadcn/ui, and Framer Motion.

## Features
- **Design**: Minimalist, premium aesthetic using shadcn/ui design system.
- **Tech Stack**: Next.js 15 (App Router), React, Tailwind CSS.
- **Animations**: Smooth page transitions and element reveals powered by Framer Motion.
- **Theming**: Dark/Light mode support.
- **Responsive**: Fully responsive mobile-first layout.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

2. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
